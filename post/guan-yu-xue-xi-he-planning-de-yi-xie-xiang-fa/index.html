关于学习和planning的一些想法

2023-03-03 3 min read

“学习是终身之事。”或许可以这样说吧，因为学习的重要性存在，所以应该时刻注意着自己对学习的思考。
而planning也是同样的重要，可以说不同的planning是不同类型的学习的配套措施。
这一篇文章也就主要讨论这两个话题。

一点闲话：
谁能想到，两周前写在纸上的想法，也会在仅仅两周之后（不到两周）就完全忘掉了，完全不知道有这样的想法了。
那么翻脸不认人的kk，恐怕也变得有点可以原谅了。
I sometimes feel tired while writing.

There is a very forward assumption that any kind of learning is the learning
of language, thus CI(Comprehendive Input) is needed.

学习的原理

学习不仅仅只是记忆，也不仅仅只是理解。

  1. 在学习的过程中，需要使用“Comprehensive Input”
  2. 学习中，专注（Focus）非常重要
  3. 并不是Design更重要，而是自己的Focus，自己的想法更重要。
  4. 学习中存在着tradeoff，是想要学得有深度的需求，和不拖延的需求的矛盾。
  5. 有人说，要掌握一项技能，只需要20h就可以到达能够使用的程度了。很多事情可以根据这个原理安排。
  6. 我认为Comprehend比Understand重要。Comprehend是有自己的理解，无论是正确还是错的。如果对所有东西做一个Instant的笔记，那么首先comes to mind的就是你自己的comprehension了。

Planning的技巧

  1. 设定Deadline
  2. 有一些Bullet Journal的页面可以买了试一下
  3. 使用Obsidian之类的productivity工具

Other Technique

  1. Have a good sleep

Some Facts

  1. 当学习了更难的东西之后，一些基础的东西和例子变得很简单，不用记很多。
  2. 其实一般解数学题的focus time挺长的。而不是一般的15min。
  3. 我感觉什么时候都需要应付Distraction这个问题
  4. 在学习的过程中，其实很难完全排除“压力”这个因素。我本人倒是很容易有压力的人。

Some thoughts

  1. In which case should you repeat?
  2. What is the difference between comprehend and understand?
  3. If we limit the focus time, what will happen?
  4. How can we learn fast?
